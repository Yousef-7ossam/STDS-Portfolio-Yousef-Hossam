# Task Description: Adventure Works Dashboard

## Project Overview
An interactive Power BI dashboard built using the Adventure Works dataset to analyze and visualize key business metrics across sales, customers, and product performance. This project demonstrates advanced data storytelling and business intelligence capabilities.

## Objectives
* **Performance Tracking:** Monitor total revenue, profit, and order trends over time.
* **Customer Analytics:** Segment customers and analyze purchasing behavior to drive marketing strategies.
* **Operational Insights:** Evaluate product category profitability and regional demand.
* **Geographical Distribution:** Visualize global sales performance through interactive mapping.
